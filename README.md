# ATM Management System

This is a simple **console-based ATM application** developed using **Core Java**. The project was created to understand and practice basic Java programming and object-oriented concepts.

The application allows a user to perform common ATM operations such as checking the account balance, depositing money, and withdrawing money using a menu-driven interface.

## Features
- Check current account balance  
- Deposit amount into the account  
- Withdraw amount from the account  
- Simple menu-based interaction  

## Technologies Used
- Java (Core Java)
- Object-Oriented Programming concepts
- Scanner class for user input

## Concepts Covered
- Interfaces and implementation
- Abstraction and encapsulation
- Polymorphism
- Control statements (loops and switch-case)
